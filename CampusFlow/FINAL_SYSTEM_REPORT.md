# Final System Report

## 1. Introduction

CampusFlow is an AI-enabled online college program management system for visitors, students, instructors, and registrars. It supports applications, course registration, course reviews, grading, complaints, graduation, semester phase management, and an AI-enabled local knowledge chatbot.

## 2. Packages and Platforms

Frontend: React, Vite, lucide-react.  
Backend: FastAPI, Uvicorn, Pydantic.  
Database: SQLite.  
Version control: GitHub.

The project uses React + FastAPI. It does not use Flask because FastAPI is already the backend web framework.
